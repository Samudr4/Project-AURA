# Overview

Project AURA is a personal AI operating system designed to serve as a secure, intelligent, and extensible assistant for software development, research, creativity, productivity, and knowledge management.

AURA combines the capabilities of:

* ChatGPT
* Claude
* Cursor
* Perplexity
* Obsidian
* Jarvis

into a unified platform while maintaining complete ownership and control over user data.

---

# Core Principles

## Privacy First

User data belongs to the user.

Sensitive information should remain protected and controllable.

---

## Intelligence First

Always utilize the best model available for each task.

---

## Human-Centered

AI assists and augments rather than replaces.

---

## Modular Design

Every subsystem should be independently replaceable.

---

## Long-Term Evolution

The platform should be able to grow without architectural rewrites.

# AI Providers

The system is designed to dynamically adapt to available API keys. It can run in Single-Provider, Dual-Provider, or Triple-Provider modes. If a key is missing, the model router dynamically mappings all tasks to the available providers.
